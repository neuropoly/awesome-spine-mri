# awesome-spine-mri
Awesome list about open-source spine MRI dataset. 

> [!WARNING]  
> If you know of spine MRI datasets not listed here, please suggest adding them by opening a [new issue](https://github.com/neuropoly/awesome-spine-mri/issues/new/choose). 

### Access legend
- 🟢 Easy: Immediate download (no approval required)  
- 🟡 Moderate: Registration or DUA required (quick access)  
- 🟠 Controlled: Application review needed  
- 🔴 Restricted: Difficult or long access process  
- ⚫ Unknown: Access conditions unclear
